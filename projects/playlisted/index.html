PLAYLISTED

Spotify playlist generator web app that allows users to create Spotify playlists based on their selected artists.

TECHNOLOGY

  • React
  • NodeJS
  • Material UI
  • Spotify Web API
  • JavaScript
  • HTML/CSS
  • Vercel

Overview

A playlist generator for Spotify users that allows users to search for Spotify artists and customize a generated playlist for them to listen to. Built on the React front-end framework with a NodeJS + Express backend for API calls to the Spotify Web API.

The idea of the project was inspired by needing an automated solution for myself when creating playlists based on music festival line ups that I attend, as I worked on the project I had decided to make a general purpose Spotify generator that others could use. I had a group of friends test the application and had decided to do a general release of the app.

This has been my most extensive project thus far incorporating both front-end and back-end development skills. This was also my first solo full-stack project.

Features of the application:

  • Allow users to login with their Spotify account to be able to automatically import generated playlists
  • Search within Spotifys library for artists
  • Customization settings for their generated playlist based on length, old vs. new tracks, include top tracks from their artist selection
  • Uses a simple algorithm to retrieve tracks from Spotify and create a playlist based on customization input
  • Responsive design to allow accessbility on all types of devices
