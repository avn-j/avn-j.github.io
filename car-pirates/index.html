CAR PIRATES

Web application for a car-sharing service allowing users book and rent cars for short periods of time, from a few hours to a few weeks.

TECHNOLOGY

  • React
  • Node.js
  • Bootstrap
  • PostgreSQL
  • TomTom API
  • Firebase
  • PayPal API

Overview

Car Pirates is a responsive web application built on the React front-end framework with NodeJS as the back-end. As part of my university capstone project, my team was assigned to build a web application based on a fictitious business idea. The delivery of the project was based on agile methodology and consisted of a team of 4 members with different roles. We worked iteratively through one-week sprints. My role on that team was a Full-Stack Developer.

The project utilised many different APIs and technologies for implementing key features. The database was ProgreSQL, the user authentication was Firebase, mock payments used Paypal API, and the maps utilised TomTom map API, and Heroku for deployments.

For this web application, I designed and developed the entire responsive user interface of the app. My responsibility was also to integrate APIs with the front end and be able to retrieve and manipulate data in the database for getting car locations and storing car information.
